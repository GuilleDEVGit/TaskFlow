# Taskflow Backend

Backend REST API built with **Spring Boot** and **MySQL** to manage users and their tasks.

This project is designed as a clean, layered backend following good practices such as DTO usage, validation, and global error handling.

---

## 🚀 Features

- User creation and retrieval
- Task creation associated with users
- Retrieve tasks by user
- DTO-based request/response handling
- Input validation with Bean Validation
- Global exception handling
- RESTful API design

---

## 🛠️ Tech Stack

- Java 17+
- Spring Boot
- Spring Data JPA (Hibernate)
- MySQL
- Maven
- Lombok
- Bean Validation
- Postman (for testing)

---

## 📂 Project Structure

- **Controller**: REST endpoints
- **Service**: Business logic
- **Repository**: Database access (JPA)
- **DTOs**: Request/response contracts
- **Exception**: Global error handling
